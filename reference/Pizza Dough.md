# Pizza Dough

Go-to recipe for woodfired pizza. Doing things by weight makes it easier to get the hydration just right.

---

## Dough (Per Ball)

| Ingredient | Weight | Notes |
|------------|--------|-------|
| Flour (00) | 350g | Caputo Classic 00 — [Mediterranean Foods, Riverside](https://www.mediterraneanfoods.nz/product/classic-00-flour-1kg/) |
| Water | 220g | ~63% hydration |
| Salt | 10g | |
| Olive oil | 18g | |
| Yeast (dry active) | 1.3g | |
| Sugar | 3.5g | |

Overnight in the fridge. Use semolina on the bottom so they don't stick to the stone.

## Oven

[Maximus Prime Schist Face 100x100cm](https://eurowoodfiredovens.co.nz/product/maximus-prime-schist-face-100-x-100cm/) — Euro Woodfired Ovens.

About **90 seconds** once the floor is around **400°C**, but it can get much hotter so keep an eye on it. Get an infrared thermometer for surface temps rather than ambient — they're cheap and make things much easier.

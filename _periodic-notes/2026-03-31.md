## ADC Pattern Extension

Adopted `docs/adc/` from the [[Agentic Context Framework]] but extended it for RPI: each ADC gets a companion directory for phase artefacts (research.md, plan.md, handoff.md). The wrapper document captures decision context — motivation, rejected alternatives, rollback strategy, risks — and points to the phase outputs. This gives agents a single entry point that tells them where they are in the process and what's been decided.

Updated the [[Agentic Context Framework]] KB note with the extension and the directory structure. Tested on the nopCommerce workshop (`docs/adc/2026-03-31--extract-cart-service`).
